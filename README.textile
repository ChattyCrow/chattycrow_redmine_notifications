h2. Installation

<pre>
cd readmine_directory/plugins/
git clone git@github.com:ChattyCrow/chattycrow_redmine_notifications.git
bundle install
rake redmine:plugins:migrate RAILS_ENV=production
</pre>

and restart redmine :)

h2. Features

h3. Sidekiq asynchronous sending

Now you don't have to wait for our servers in redmine main thread, easily add Sidekiq to your redmine project. You can easilly add sidekiq plugin, start it and allow sidekiq in redmine chatty crow notification plugin settings.

<pre>
https://github.com/ogom/redmine_sidekiq
</pre>

and it's done :).
